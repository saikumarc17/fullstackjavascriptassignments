What is HTML?

HTML (HyperText Markup Language) is the code that is used to structure a web page and its content. HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content to make it appear a certain way, or act a certain way.

What is semantic HTML

semantic tags are elements that provide meaning and structure to the content of a web page. They go beyond just defining the appearance or formatting of the content and help search engines and assistive technologies understand the purpose and hierarchy of the information. Semantic tags were introduced in HTML5 to promote better accessibility, search engine optimization (SEO), and code readability

Commonly used sementic tags

  1. `<header>`: Represents the introductory content or a container for a group of introductory content at the top of a document or section.
  2. `<nav >`: Defines a section that contains navigation links.
  3. `<main>`: Specifies the main content of a document or the main content within an `article` element.
  4. `<section>`: Defines a standalone section that logically groups related content.
  5. `<footer>`: Represents the footer or closing section of a document or a section.
