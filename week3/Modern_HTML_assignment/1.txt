Some of the features of HTML5 are :-
1. Semantics: HTML5 introduces new semantic elements such as `<header>`, `<nav>`, `<article>`, `<footer>`, and `<section>`, which provide clearer structure and meaning to web content.

2. Multimedia Support: HTML5 includes native support for audio and video playback without the need for third-party plugins like Flash. 
The `<audio>` and `<video>` elements allow embedding media directly into web pages.

3. Canvas: The `<canvas>` element allows dynamic rendering of graphics, animations, and interactive visualizations using JavaScript. 
It provides a powerful API for drawing shapes, images, and text directly in the browser.

4. Geolocation: HTML5 enables websites to request and access the user's location information through the Geolocation API. 
This feature allows applications to provide location-based services and personalize content based on the user's geographical position.

5. Local Storage: HTML5 introduced the `localStorage` and `sessionStorage` APIs, which allow web applications to store data locally on the user's device. 
This enables offline access, faster loading times, and persistent data storage for web applications.

6. Drag and Drop and copy to clipboard: HTML5 provides built-in support for drag and drop operations.
 With the Drag and Drop API, developers can create intuitive interfaces for users to drag elements and drop them into specific targets.
 With copy to clipboard user can easily copy the text to clipboard.

7. Forms and Input Types: HTML5 introduced new input types like `date`, `email`, `number`, `range`, `search`, and `url`, which provide better input validation and user experience for specific data types. It also includes new form elements like `datalist` etc.
