Text Formating Tags in HTML

  1. bold tag - <b>
    • It is used to make text bold
    • It is a presentational element that does not carry any semantic meaning
    • Don't use for bold tag for entire paragraph
    • It is recommended to use it sparingly and only when necessary

  2. Paragraph tag - <p></p>

    • It is a block level element
    • It is mostly used in discribing big paragraphs
  3. mark tag - <mark></mark>
    • mark element represents a run of text in one document marked or highlighted for reference purposes
    • it indicates a highlight
